Kotlin Slide Puzzle (Swing)KotlinとJava Swingで構築された、クラシックな3x3スライドパズルゲームです。
ランダムにシャッフルされた数字を並び替え、1から8までの順番を完成させてください。
🎮 ゲームの特徴直感的な操作: 数字ボタンをクリックするだけで、空きスペースに隣接していればパネルがスライドします。
動的シャッフル: 起動するたびにパネルがランダムに配置されます。クリア判定: 全ての数字が正しい位置（1〜8の順）に並び、最後が空きマスになるとクリアメッセージが表示されます。軽量設計: 外部ライブラリに依存せず、標準のJava/Kotlin環境のみで動作します。
🚀 実行方法前提条件JDK 8 以上Kotlin コンパイラ (kotlinc)コンパイルと実行リポジトリをクローンした後、以下のコマンドを実行してください。
Bash# コンパイル
kotlinc SlidePuzzleSwing.kt -include-runtime -d SlidePuzzle.jar

# 実行
java -jar SlidePuzzle.jar
🛠 技術的な仕組み
1. グリッド制御GridLayout(3, 3) を使用して、ボタンを規則正しく整列させています。
2. 2. 隣接判定ロジッククリックされたボタンが空きマス（Blank）に移動可能かどうかを、以下の数式を用いて判定しています。
3. インデックスから2次元座標 $(x, y)$ を算出し、その差の絶対値の合計が 1 であるかどうかを確認します。
4. $$|x_1 - x_2| + |y_1 - y_2| = 1$$3. クリアチェック現在のボタンのテキスト配列と、目標となる配列 ["1", "2", "3", "4", "5", "6", "7", "8", ""] を比較することで勝利判定を行っています。
5. 📂 コード構成initブロック: コンポーネントの初期化、数字のシャッフル、UIへの配置。handleMove: クリックイベントの処理とテキストの入れ替え。
6. isAdjacent: 隣接チェックのアルゴリズム。checkWin: 勝利条件の検証。
